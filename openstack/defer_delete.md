#### glance镜像的延迟删除
+ 用法  
glance-api.conf中打开延迟删除开关
``` xml
# Turn on/off delayed delete
delayed_delete = False
 
# Delayed delete time in seconds
scrub_time = 43200
```
删除镜像fedora_s2。
